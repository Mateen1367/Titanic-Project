# Titanic-Project
Using Titanic Dataset to analyze survival and death causes backed by years of research article backing our analysis 
Project Overview

This project uses the Titanic dataset to analyze patterns of survival and death among passengers. The goal is to identify key factors that influenced survival rates and support findings using data analysis and research-backed reasoning.

We will focus on exploring how characteristics such as gender, age, passenger class, and socioeconomic status affected survival outcomes. Initial observations suggest that women and children had higher survival rates, and this project will investigate why that occurred through data filtering, visualization, and comparison.

Objectives:
-Identify which groups of passengers were more likely to survive
-Analyze survival differences based on gender, age, and class
-Explore possible reasons behind these patterns using data and research
-Visualize survival trends using charts and graphs
-Compare survival vs. death cases to understand key contributing factors

Analysis Approach:
-Data Cleaning: Handle missing values (age, cabin, etc.) and prepare dataset
-Filtering: Separate survivors vs. non-survivors for deeper comparison
-Feature Analysis:
-Gender (male vs female survival rates)
-Age groups (children vs adults)
-Passenger class (1st, 2nd, 3rd class)
-Visualization: Create bar charts, histograms, and correlation plots to show trends
-Research Support: Compare findings with historical explanations (e.g., “women and children first” policy)

Key Questions:
Did women and children actually survive more?
How much did passenger class affect survival chances?
Were wealthier passengers more likely to survive?
What patterns can we find among passengers who did not survive?
